# Portafolio Educación Inclusiva – Global Schools Prize

Este repositorio organiza una maqueta de portafolio para la categoría **Educación Inclusiva** del Global Schools Prize,
basado en el trabajo del Liceo Bicentenario de Excelencia Polivalente San Nicolás.

Incluye:

- Página principal con ejes de trabajo inclusivo, marco normativo y proyectos emblemáticos.
- Páginas internas para PIE, DUA, proyectos XR, familias, contexto y mapa de evidencias.
- Barra de controles inclusivos (tema, tamaño de letra, narrador, idioma ES/EN y buscador interno).
- Estructura lista para ser completada con evidencias reales, enlaces y documentos del liceo.
